ğŸ“˜ What is Docker?

Docker is an open-source platform used to build, ship, and run applications in isolated environments called containers.

ğŸ”§ Basic Docker Commands

# Check Docker version
docker --version

# Pull nginx image
docker pull nginx

# Run container
docker run -d -p 8080:80 nginx

# List running containers
docker ps

# Stop container
docker stop <container_id>

# Remove container
docker rm <container_id>

ğŸŒ Create a Website Using Docker

Steps to run an HTML page inside a Docker container:

  1. Create your index.html
  2. Write a simple Dockerfile:
FROM nginx:alpine
COPY index.html /usr/share/nginx/html/index.html

Build and run:

docker build -t my-webpage .
docker run -d -p 8080:80 my-webpage

ğŸ’¡ Tips for Docker Usage

  • Always use lightweight images like alpine if possible
  • Tag images properly (e.g., my-webpage:v1)
  • Use .dockerignore to avoid copying unnecessary files
  • Use Docker Compose for multi-container setups
